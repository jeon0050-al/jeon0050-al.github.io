  1. Qus: What are the three components of Web?

    Ans: Ans for Qus 1 HTML, CSS, JavaScript

  2. Qus: Define version control and give one example.

    Ans: Ans for Qus 2 Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later. e.g.GitHub

  3. Qus: What tag is used to create the title of a web page? Give an example.

    Ans: Ans for Qus 3 <title> tag e.g. <title>Basic HTML Tags</title>

  4. Qus: What tag is used to create a paragraph in html? Give an example.

    Ans: Ans for Qus 4 <p> tag e.g. <p>Paragraph in the main tag</p>

  5. Qus: Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?

    Ans: Ans for Qus 5 <p> tag, reason: a <p> tag is an block element while the rest of tags are inline elements.

  6. Qus: What are the tags required to create a navigation list in HTML?

    Ans: Ans for Qus 6 <li>tag

  7. Qus: What element is used to create images? Name its attributes.

    Ans: Ans for Qus 7 <img>tag, attributes are src, width, height, alt

  8. Qus: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason

    Ans: Ans for Qus 8 <b>tag, the reason is <b> tag specifies bold text and the rest of them are used to make a table.

  9. Qus: Give an example of color picker input.

    Ans: Ans for Qus 9 <input type="color" name="co" id="co">

  10. Qus: What is the use of name and value attributes? Give two examples

    Ans: Ans for Qus 10 The name attribute specifies a name for an HTML element, and it can be used to reference the element in a JavaScript.
    The value attribute saves the input and submits it to the server, and it is permanent and would not disappear when the user starts inputting.
    <input type="email" name="email" id="email" value="example@g.com">
    <select name="province" id="province">
    <option value="AB">Alberta</option>

  12. Qus: How would you link an external style sheet? Give example.

  13. Qus: Create CSS block to center an element both horizontally and vertically that has position:absolute and has width:200px and height:200px.

    Ans: Ans for Qus 13
    .element {
    position: absolute;
    left: 50%;
    top: 50%;
    width: 200px;
    height: 200px;
    margin: -75px 0 0 -75px;
    }

  14. Qus: Create CSS keyframes block called "spin" that rotates an element.

    Ans: Ans for Qus 14
    @keyframes spin {
    from {
    transform: rotate(0deg);
    }
    to {
    transform: rotate(360deg);
    }
    }
